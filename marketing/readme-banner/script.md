# 2-minute explainer — VO script

Total target: ~115 sec @ ~150 wpm = ~290 words. Punctuation-driven pacing
for natural pauses. "S H" spelled phonetically so the model says letters
rather than "sh" sound.

---

We open documents in apps. Word for dot D O C X. Excel for dot X L S X.
PDF for PDF. Each one needs the right tool.

But every device on earth already has a universal viewer — the browser.
So... what if a document was just an HTML file?

This is a workbook. A plain HTML file. It opens on any device. No app
to install. Nothing to sign up for. It just opens.

Because it's HTML, it can do anything HTML can do. Show charts. Run
machine learning in WebAssembly. Talk to data sources. Behave like a
full application. While staying one file.

The one thing browsers don't do is save files in place. So we built a
tiny background program — about a megabyte — that gives your browser
save in place. Install once at workbooks dot S H, and every workbook
on your machine saves like a document.

Author your own with the workbook command line tool. Init. Dev. Build.
Out comes one dot workbook dot HTML file. Email it. Send it. Drop it on
a USB stick. It opens for whoever receives it.

Documents you can use. Applications you can email. One universal viewer
everyone already has. That's a workbook.

Workbooks dot S H.
