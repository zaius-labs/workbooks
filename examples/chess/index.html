Chess between two browsers

No server, no account, no copy-paste. The first browser to open this file becomes the host, generates a peer ID, and turns the URL into a join link. Open that link in another browser — same machine, same network, anywhere on the internet — and the WebRTC data channel forms in seconds. Move state syncs over the channel directly; the public PeerJS broker only sees the SDP exchange.
